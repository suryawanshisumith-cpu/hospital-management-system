# hospital-management-system
A simple Hospital Management System developed in C programming language.
This project helps manage patient records, doctor assignments, room allocation, medicines, and discharge details using basic Data Structures and File Handling concepts.

FEATURES -
1.Add Patient Records
2.View Patient Details
3.Search Patient by ID
4.Assign Doctors
5.Room Allocation System
6.Medicine Management
7.Patient Discharge System
8.Doctor Availability Check
9.File Handling for Data Storage
8.Menu Driven Program

TECHNOLOGIES USED -
.C Programming
.Structures
.Functions
.Arrays
.File Handling
.Project Purpose

-File handling in C
